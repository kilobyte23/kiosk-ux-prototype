# Project Overview

## Purpose
High-fidelity UX prototype for a **2-wheeler EV battery-swapping kiosk**, built to demonstrate complete interaction flows before engineering development begins.

## Audience
- **Stakeholders** — Product behavior review and sign-off
- **Designers** — Screen flow creation and iteration
- **Engineers** — Production UI architecture reference

## User Groups
1. **Customers** — EV riders swapping batteries at the kiosk
2. **Admins** — Dealer staff, technicians, and OEM engineers managing the station

## Tech Stack
- React + TypeScript + Vite (prototype phase)
- Designed for future migration to production stack
