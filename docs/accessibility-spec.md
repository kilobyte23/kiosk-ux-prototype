# Accessibility Specification

## WCAG 2.1 Compliance Target: AA

### Touch Targets
- Minimum: 48 × 48 px
- Comfortable: 56 × 56 px
- Glove/Rain mode: 72 × 72 px
- No fine motor precision required

### Visual
- High contrast mode (3px outlines, #FFF on #000)
- Large text mode (all tokens × 1.5)
- Sunlight / anti-glare mode (boosted contrast + reduced transparency)
- Automatic brightness adjustment

### Audio
- Voice guidance mode for visually impaired users
- Optional sound cues (toggle-able)
- No audio-only information

### Motor
- One-hand operable
- No gesture complexity above single tap / swipe
- Rain / glove mode with increased touch sensitivity

### Session
- 60-second persistence
- 15-second timeout warning
- All actions reversible within session
