# Local Reporting & Logs — Flow

## Screens
1. **SwapTransactionLogs** — Offline logs + CSV export
2. **EnergyConsumption** — Daily / weekly reports
3. **BatteryRotationAnalytics** — Most-used vs least-used
4. **UptimeReporting** — Planned vs unplanned downtime
5. **ErrorFrequencyHistogram** — Top recurring fault codes
6. **EndOfDaySummary** — Auto email to station manager
7. **LogRetentionConfig** — 30 / 60 / 90 days
8. **SyslogExport** — Remote syslog integration
