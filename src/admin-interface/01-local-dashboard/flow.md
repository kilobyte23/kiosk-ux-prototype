# Local Admin Dashboard — Flow

## Screens
1. **DashboardOverview** — Landing page, system summary
2. **SystemResourceMonitor** — CPU / RAM / Storage / Threshold alerts
3. **ProcessHealthPanel** — Watchdog: BMS, Payment, Display, Comms
4. **HardwareIOMonitor** — Slot relays, door locks, sensors
5. **LogViewer** — Severity filter, keyword search
6. **FirmwareVersionPanel** — Update trigger, rollback
7. **MaintenanceScheduler** — Visual downtime calendar
8. **FactoryResetWizard** — Passphrase confirmation

## Access
- Local web dashboard on LAN (localhost), no internet dependency
