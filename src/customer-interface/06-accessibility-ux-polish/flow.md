# Accessibility & UX Polish — Flow

## Screens
1. **HighContrastScreen** — High contrast mode demo
2. **LargeTextScreen** — Large text mode demo
3. **RainGloveScreen** — Larger buttons, increased touch sensitivity
4. **SunlightAntiglareScreen** — Anti-glare mode demo

## Design Requirements
- Minimum 48×48px touch targets
- No fine motor precision required
- Voice guidance mode for visually impaired
- Automatic brightness adjustment
- Session persistence for 60 seconds
- Animated loading indicators
- Consistent navigation controls
